Walmart Sales Analysis
Project Overview

This project analyzes Walmart sales data using Python for Exploratory Data Analysis (EDA) and business insights generation.

The objective of this project is to identify:

Sales trends
Profit analysis
Best performing categories
Regional performance
Customer purchasing behavior
